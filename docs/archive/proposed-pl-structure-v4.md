## COGS

### Services COGS

**Managed Services**
- Labor
  - W-2
  - Outsourced / Offshore
- Software (Agreement Licenses, e.g., M365, Barracuda, Backup)
- Tools (Internal HD tools, e.g., Skout, Kaseya, CW)

**Consulting/Projects**
- Labor
  - W-2 (Engineering + COD)
  - Outsourced / Subs
- Software (if any direct project software)
- Tools (if any direct project tools)

**Comprehensive Services**
- Labor
  - W-2 (if any management overlay)
  - Outsourced / Dedicated Subs (e.g., Blue Pisces, Fredrick)

**Other**
- Rebates/Incentives
- Miscellaneous

### Products COGS
- Hardware
- Software (Resale purchases not bundled into MS)
