## Three-Question Audit Process

For every tool in the "Evaluate" category, answer these three questions:

**Q1: Is it actively used?**
- Check: Login frequency in the last 90 days
- Check: Ask the team member who owns it
- If no active use in 90 days → Cancel

**Q2: Does it duplicate another tool?**
- Scheduling: TimeZest is the standard. Anything else is redundant.
- AI/LLM: Pick one primary API (OpenAI or Anthropic). Cancel the others.
- Storage: Wasabi is primary. Cancel everything else.
- Project management: ConnectWise is the standard. Monday.com needs justification.
- If it duplicates → Cancel the lower-value tool

**Q3: Is it client-billable?**
- If yes → Move to COGS, not SGA
- If no → Apply stricter justification standard
